In-memory implementation of IEventStore for testing and development purposes. Provides thread-safe event storage with multi-tenancy support and comprehensive functionality.

In-memory implementation of ISnapshotStore for testing and development purposes. Provides thread-safe snapshot storage with multi-tenancy support.

IntervalSnapshotStrategy

A snapshot strategy that takes snapshots at regular intervals based on event count.

A snapshot strategy that never takes snapshots. This is the default strategy when no snapshot store is configured.
