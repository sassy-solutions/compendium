An AI agent: takes a user prompt + a tool registry, runs a multi-turn loop with the underlying IAIProvider, and returns the final answer plus a turn-by-turn audit trail.

public interface IAgent

Remarks

The default implementation (StandardAgent in Compendium.Application) uses ReAct-style prompt injection — tool descriptions and an action grammar are rendered into the system prompt, and the agent parses an ```action block out of the model's response to decide whether to invoke a tool. This keeps the agent layer decoupled from any provider's native tool-calling format.

Turns is the audit trail you persist for observability, debugging, and replay. Each turn carries the model output, tool invocations, usage stats, and latency.
