Class ReActPromptBuilder

Namespace
Compendium.Application.AI.Agents
Assembly
Compendium.Application.dll

Builds the system prompt that turns a vanilla LLM into a tool-using ReAct agent. The model is told to emit an ```action JSON block when it wants to call a tool; the agent parses that block, dispatches to the registry, feeds the result back, and loops. Public so callers can build custom agents that share the same grammar.
