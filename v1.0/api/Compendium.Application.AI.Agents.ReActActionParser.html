Parses an action ... block out of an assistant message. Tolerant of surrounding prose and of the model's preferred whitespace conventions; intolerant of malformed JSON inside the block (we surface those as parse errors rather than guessing). Public so callers can build custom agents that share the same grammar.

TryParse(string, out ParsedAction?, out string?)

Tries to extract an action from content. If no action block is present, the method returns false with both action and parseError set to null.

public static bool TryParse(string content, out ReActActionParser.ParsedAction? action, out string? parseError)

Parameters

content string

The raw assistant message content.

action ReActActionParser.ParsedAction

The parsed action when the method returns true; otherwise null.

parseError string

A user-facing description of the parse failure when the method returns false for a malformed block; otherwise null.

Returns

bool

true when an action block was found and successfully parsed; otherwise false. When this method returns false, parseError is non-null if a block was present but malformed, and both action and parseError are null if no action block was found at all.
