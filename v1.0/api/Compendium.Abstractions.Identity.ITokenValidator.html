Interface ITokenValidator

Namespace
Compendium.Abstractions.Identity
Assembly
Compendium.Abstractions.Identity.dll

Provides operations for validating and introspecting tokens. This interface supports both local JWT validation and remote token introspection.

public interface ITokenValidator

Methods

IntrospectTokenAsync(string, CancellationToken)

Introspects a token by calling the identity provider's introspection endpoint. This method provides real-time validation status and is more accurate than local validation for checking if a token has been revoked.

Task<Result<TokenIntrospectionResult>> IntrospectTokenAsync(string token, CancellationToken cancellationToken = default)

Parameters

token string

The token to introspect.

cancellationToken CancellationToken

The cancellation token.

Returns

Task<Result<TokenIntrospectionResult>>

A result containing the introspection result or an error.

ValidateTokenAsync(string, CancellationToken)

Validates a token and extracts its information. This method typically performs local validation using the issuer's public keys.

Task<Result<TokenInfo>> ValidateTokenAsync(string token, CancellationToken cancellationToken = default)

token string

The token to validate (typically a JWT).

cancellationToken CancellationToken

Returns
