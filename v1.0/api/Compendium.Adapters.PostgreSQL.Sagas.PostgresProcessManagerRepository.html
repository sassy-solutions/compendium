PostgreSQL implementation of IProcessManagerRepository. Persists process-manager metadata, a JSON snapshot of state (best-effort, optional), and the per-step status table. Multi-tenant aware via ITenantContext.

Remarks

The contract does not currently expose a version/ETag on IProcessManager, so optimistic concurrency cannot be expressed at this layer; durable adapters are nonetheless encouraged to track an internal version column and surface conflicts as Conflict(string, string, IReadOnlyDictionary<string, object>?) when they detect concurrent writes. A future minor version may extend the abstraction with an explicit expectedVersion argument.
