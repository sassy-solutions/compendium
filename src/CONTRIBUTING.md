# Contributing to Compendium

This directory is the **staging area** for Compendium inside the Nexus monorepo.
The framework will be extracted to its own repository (`sassy-solutions/compendium`)
in a follow-up milestone (see POM-146 / POM-149).

Until the extraction lands, contributions are welcome via the main Nexus
repository:

- Open issues and pull requests against `sassy-solutions/Nexus`.
- Scope PRs that touch `src/Framework/` to framework-only changes when possible.
- Follow the existing code style (file headers, .editorconfig, Result pattern).

Once the standalone repository exists, this file will be replaced with the
full contribution guide (code style, commit conventions, release flow).
