Defines the contract for executing and compensating individual process manager steps. Implementations issue commands to aggregates / external systems and report success or failure back to the orchestrator.

CompensateAsync(IProcessManager, SagaStep, CancellationToken)

Compensates (undoes) a previously executed step. Compensations should be idempotent: the orchestrator may retry on transient failures.
