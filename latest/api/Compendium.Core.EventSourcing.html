Namespace Compendium.Core.EventSourcing

Classes

EventTypeMetadata

Metadata about a registered event type.

EventTypeRegistry

Thread-safe registry for whitelisted event types using .NET 9 frozen collections for optimal performance. Prevents deserialization attacks by maintaining a strict whitelist of allowed domain event types.

EventUpcasterBase<TSource, TTarget>

Abstract base class for event upcasters. Provides common implementation of the non-generic IEventUpcaster interface.

EventVersionMigrator

Manages event version migrations using registered upcasters. Applies a chain of upcasters to migrate events from old versions to the latest schema.

SecureEventDeserializer

Secure event deserializer that prevents deserialization attacks by using a whitelist-based approach. Only registered domain event types can be deserialized, preventing arbitrary code execution. Automatically applies event version migrations using registered upcasters.

Interfaces

IEventDeserializer

Secure event deserializer that prevents deserialization attacks using whitelisted types.

IEventTypeRegistry

Registry for whitelisted event types to prevent deserialization attacks.

IEventUpcaster

Non-generic interface for event upcasters. Used for registration and discovery of upcasters at runtime.

IEventUpcaster<TSource, TTarget>

Interface for upcasting events from an old schema version to a new version. Event upcasters enable schema evolution while maintaining backward compatibility.

IEventVersionMigrator

Interface for the event version migrator service.

IGeneratedEventRegistry

Interface for generated event registry implementations. Source generators implement this interface to provide compile-time safe event registration.
