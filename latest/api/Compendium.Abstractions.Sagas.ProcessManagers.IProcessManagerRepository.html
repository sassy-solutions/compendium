Retrieves a process manager by id, deserializing its persisted TState snapshot. Required for resuming a saga from any step with full typed state access — e.g. to detect already-completed external work and stay idempotent.

Task<Result<IProcessManager<TState>>>

A successful result with the process manager and rehydrated state, a not-found error, or a deserialization error if the persisted state cannot be decoded into TState. If no state has been persisted yet, a default-constructed TState is returned.

Remarks

The contract does not currently expose a version/ETag on IProcessManager, so optimistic concurrency cannot be expressed at this layer; durable adapters are nonetheless encouraged to track an internal version column and surface conflicts as Conflict(string, string, IReadOnlyDictionary<string, object>?) when they detect concurrent writes. A future minor version may extend the abstraction with an explicit expectedVersion argument.
