BeginEventSourcingScope(ILogger, string, string?, string?, long?)

Creates a logging scope with event sourcing context that enriches all logs within the scope.

public static IDisposable BeginEventSourcingScope(this ILogger logger, string aggregateId, string? tenantId = null, string? eventType = null, long? aggregateVersion = null)

Parameters

logger ILogger

The logger.

aggregateId string

The aggregate ID.

tenantId string

The tenant ID (optional).

eventType string

The event type (optional).

aggregateVersion long?

The aggregate version (optional).

Returns

IDisposable

A disposable scope.

Examples

using (_logger.BeginEventSourcingScope("user-123", "tenant-456", "UserCreated", 1))
{
    _logger.LogInformation("Processing user creation event");
    // All logs in this scope will include AggregateId, TenantId, EventType, AggregateVersion
}
