Class SecureEventDeserializer

Namespace
Compendium.Core.EventSourcing
Assembly
Compendium.Core.dll

Secure event deserializer that prevents deserialization attacks by using a whitelist-based approach. Only registered domain event types can be deserialized, preventing arbitrary code execution. Automatically applies event version migrations using registered upcasters.
