Remarks

IIntegrationEvent exposes CorrelationId and CausationId as read-only properties, so callers must populate them on the event being constructed — typically by reading CorrelationId and CausationId from this context. Custom publishers that prefer envelope-based propagation can override this contract.

PublishCompensationAsync<TEvent>(TEvent, CancellationToken)
