Parameters

Subject string

The subject under evaluation (typically user:<id> or a userset reference).

Relation string

The relation to check, as defined in the authorization model.

Object string

The object the relation is checked against (typically <type>:<id>).

TenantId string

The tenant whose authorization store should answer the check.

ContextualTuples IReadOnlyList<RelationTuple>

Optional in-flight tuples added to the evaluation context but not persisted to the store. Use for "what-if" evaluations or for relationships that exist only within the request scope.
