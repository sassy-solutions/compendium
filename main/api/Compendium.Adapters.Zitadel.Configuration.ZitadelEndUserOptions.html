Remarks

This configuration is separate from ZitadelOptions because End-Users use a different Zitadel Project and Application than PlatformUsers.

End-Users:

  • Self-register through Zitadel
  • Are scoped to specific tenants and applications
  • Have different permission structures (app-based vs platform-based)
