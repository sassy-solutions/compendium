Parameters

Language string

Optional BCP-47 language hint (for example en-US, fr-FR). When null, the provider is expected to auto-detect.

Model string

Optional provider-specific model identifier (for example whisper-large-v3, nova-3).

Diarization bool

When true, the provider should label speaker turns in the output. Defaults to false.

Punctuation bool

When true, the provider should insert punctuation and capitalisation. Defaults to true.
