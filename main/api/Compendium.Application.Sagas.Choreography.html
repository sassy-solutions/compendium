Namespace Compendium.Application.Sagas.Choreography

Classes

ChoreographyRouter

Default IChoreographyRouter that resolves all registered IHandle<TEvent> implementations from DI and dispatches the event to each. Failures from individual handlers are aggregated into a single error.

InMemoryIntegrationEventPublisher

In-memory IIntegrationEventPublisher intended for tests, samples, and quick prototypes. Buffers published events for inspection and optionally re-routes them through a IChoreographyRouter for end-to-end fan-out.
