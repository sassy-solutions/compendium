Class RedisIdempotencyStore

Namespace
Compendium.Adapters.Redis.Idempotency
Assembly
Compendium.Adapters.Redis.dll

Redis implementation of IIdempotencyStore for distributed idempotency tracking. Provides high-performance, distributed storage for operation results with configurable TTL. All infrastructure failures (Redis connection, serialization, timeouts) are surfaced via Result<TValue> failures rather than exceptions, matching the Compendium Result pattern.
